# Sales Goal Calculator

A command-line tool to track sales performance against targets. Built with Python as a practical project to consolidate fundamentals: loops, conditionals, data structures, input validation, and file I/O.

## What it does

- Calculates how much is left to hit the sales target
- Shows percentage of goal achieved with motivational feedback
- Estimates the daily sales needed to reach the target in the remaining days
- Keeps a session history of multiple calculations
- Exports a summary report to `historico.txt` at the end of the session

## How to run

Requires Python 3.

```bash
git clone https://github.com/Marciolpedroso/calculadora-metas-vendas.git
cd calculadora-metas-vendas
python script.py
```
